# QKart Frontend

## 🚀 Description
QKart is an e-commerce web application where users can browse products, search items, add products to cart and checkout.

## 🛠️ Tech Stack
ReactJS  
JavaScript  
Material UI  
REST API  

## 🎯 Features
• Browse products  
• Product search  
• Add to cart  
• Checkout system  
• Responsive UI  

## 💻 How to Run

1. Clone the repo
git clone https://github.com/sonu-organizein/ME_QKART_FRONTEND_V2.git

2. Install dependencies
npm install

3. Run the project
npm start
